# Bybit Adaptive Trading Bot

Робот для автоматизированной торговли на Bybit (Unified Trading Account) с использованием адаптивного выбора стратегий.

## Основные возможности
- **Три стратегии параллельно:** Пробой уровней, Отскок от уровней, Движение по тренду.
- **Виртуальное тестирование:** Бот постоянно тестирует все стратегии в "песочнице" и выбирает наиболее прибыльную для реальных торгов.
- **Риск-менеджмент:** 
  - Плечо: 5x.
  - Депозит $90 разделен на 3 слота по $30.
  - "Kill Switch": Остановка торговли при серии из 5 убытков подряд.
- **Логирование в БД:** Полная история сделок и профита сохраняется в SQLite.
